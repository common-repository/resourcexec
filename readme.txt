A lightweight, easy to use plugin to schedule people, equipment and other resources. 

== Description ==

ResourcExec is intuitive, easy to use online resource management WordPress plugin that requires little to no training. 

Get your resource scheduling under control with a fast, simple WordPress plugin solution. 

###Features###
* __Fast & Flexible Scheduling.__ 
Add, edit, move and delete bookings in seconds. Any duration you need: add bookings that stretch from 5 minutes to weeks. Any type of resources - people, meeting rooms, equipment. Assign multiple resources to one booking.

* __No Conflicts.__
Clashes are alerted and can be automatically prevented.

* __Visual Calendar.__
The schedule view gives you a clear picture of who is working on what. 

* __Reports.__
Monitor utilization for all your resources at a glance or focus on specific individuals. Set custom date ranges.

Get started in minutes with the [quick start guide](http://www.resourcexec.com/documentation/) now!

== Installation ==
= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. 
To do an automatic install:

* Log in to your WordPress admin panel, navigate to the Plugins menu and click _Add New_.
* In the search field type "ResourcExec" and click _Search Plugins_.
* Once you've found the plugin you can install it by clicking _Install Now_.

= Manual Installation =
The manual installation method involves downloading the plugin and uploading it to your web server.

* Download the plugin file and unzip it.
* Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's _wp-content/plugins/_ directory.
* Activate the plugin through the 'Plugins' menu in the WordPress admin.
